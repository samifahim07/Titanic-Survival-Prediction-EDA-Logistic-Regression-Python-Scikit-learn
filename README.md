# Titanic-Survival-Prediction-EDA-Logistic-Regression-Python-Scikit-learn
End-to-end ML project on the Titanic dataset. Covers data loading, EDA (countplots, heatmaps, correlation), missing value handling (mean/bfill), label encoding, and Logistic Regression classification. Achieves ~81.6% test accuracy. Built with Pandas, Seaborn, Matplotlib &amp; Scikit-learn.
