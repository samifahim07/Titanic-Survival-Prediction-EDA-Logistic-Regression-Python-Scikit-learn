**🚢 Titanic Survival Prediction | End-to-End Machine Learning Project**

This project presents a complete machine learning workflow using the Titanic dataset to predict passenger survival. It begins with thorough data cleaning and preprocessing, including handling missing values, treating inconsistent entries, and preparing categorical features through encoding techniques.

Exploratory Data Analysis (EDA) is performed to uncover meaningful patterns and relationships between features such as gender, passenger class, age, and fare. Visualizations are used to highlight how these factors influenced survival rates.

Feature engineering techniques are applied to improve model performance, followed by training a Logistic Regression model. The model is evaluated using multiple performance metrics including accuracy (~85%), cross-validation score, ROC-AUC, classification report, and confusion matrix to ensure robustness and reliability.

**🔧 Tech Stack:**
Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

**📊 Key Highlights:**

* Data cleaning & preprocessing
* Handling missing and noisy data
* Feature encoding & transformation
* Insightful EDA with visualizations
* Model building using Logistic Regression
* Performance evaluation with multiple metrics
